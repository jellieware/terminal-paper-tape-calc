# terminal-paper-tape-calc
A paper tape calculator for the terminal
<br>
<br>
Usage example:
<br>
<br>
Add 500: +500<br>
Delete ID # 3: d3<br>
5%: p5<br>
-5%: m5<br>
exit: x
