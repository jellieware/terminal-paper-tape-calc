# terminal-paper-tape-calc
A paper tape calculator for the terminal
<br>
<br>
Usage example:
<br>
<br>
Add 500: "+500"<br>
Divide by 6: "d6"<br>
Subtract by 700: "-700"<br>
Multiply by 4: "*4"<br>
Delete ID # 3: "e3"<br>
5%: "p5"<br>
-5%: "m5"<br>
Print to file: "f"<br>
Clear All: "c"<br>
Exit: "x"
